# 机器学习资源整理

## 背景介绍

本项目是个人学习的资源和代码整理，包含本人学习过程中所用的资料以及代码实现，目的一是对学习的内容进行整理和归纳，方便今后的查阅和复习，
二是将自己的学习过程分享出来，和大家一起学习，共同进步，有任何问题和建议欢迎联系**sunyangwei@ict.ac.cn**.
